# README.md

This is my personal dotfiles repository, where I keep custom configurations for my working environment.
It includes configurations for my command-line tools, text editors, and more.

### What are dotfiles?

Dotfiles are configuration files used to customize the working environment on Unix-based systems,
such as macOS and Linux. These files have names starting with a dot (.) and are located at the root
of the user's home folder. Some examples of dotfiles are `.bashrc`, `.vimrc`, and `.gitconfig`.

## Included Configurations

- `Alacritty`: A cross-platform, OpenGL terminal emulator..
- `Dmenu`: Suckless's  dynamic menu.
- `Dunst`: Lightweight notification manager.
- `Dwm`: Suckless's dybamic window manager (include dwmblocks).
- `Mpv`: Free and open-source media player.
- `Neovim`: Vim-based text editor.
- `Ranger`: Cli file manager.
- `Redshift`: Color temperature tool.
- `Qtile`: Hackable tiling window manager written in Python.
- `Slock`: Suckless's simple locker.
- `Zathura`: Lightweight document viewer.

## Some useful scripts

* `dmemoji`: Emoji selector using dmenu.

* `dmkill`: Processes killes using dmenu.

* `dmpower`: Power menu using dmenu.

* `screenshot`: Script for taken screenshot using scrot.

## How to use these dotfiles

1. Clone this repository to your local computer.
2. Copy the files you need to the corresponding location on your system. For example, if you want to use
   my `Qtile` configuration, copy the folder to your home folder (`~/.config/qtile`).
3. Make sure the files are visible on your system. Files and folders starting with a dot are hidden by default,
   so you may need to enable the option to show hidden files in your file explorer.
4. Customize the files according to your needs. Remember that these are my personal dotfiles and may
   not be suitable for your specific needs.
